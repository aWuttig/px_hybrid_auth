What does it do?
^^^^^^^^^^^^^^^^

The **HybridAuth** extension (*px_hybrid_auth*) is used to authenticate frontend users against a social provider (e.g. Facebook, LinkedIn) mostly via OAuth.
It uses HybridAuth (http://hybridauth.sourceforge.net/) as an abstract api to social apis and identities providers such as Facebook, Twitter, MySpace and Google.

- **Login Form:** It brings up a new content element »HybridAuth Login Form« to the content element wizard on the »Form elements« tab. It includes several methods for redirecting after login\logout as well as felogin extension.
- **Plugin:** The extension also includes a plugin to list the connection state of the social accounts from some frontend user.

Supported social providers
^^^^^^^^^^^^^^^^^^^^^^^^^^

The following social providers are supported by the extension:

- Facebook
- LinkedIn
- Xing
